# Math 

Currently serves only one purpose. To calculate your missing sides of a triangle.
It also calculates the angles inside of that triangle.

# Features
- Solve right triangles (Pythagorean theorem + angles)
  - It asks you to place the triangles and uses OOP to filter your input.


# Installation / How to run
1. Go to the master branch
2. Download the zip file or only the math.py file.
2. Unzip the file or open math.py
3. Then the pythagorean theorem starts automatically.
